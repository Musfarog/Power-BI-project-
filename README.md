# 📊 Power BI Data Analysis Project

Shows my ability to use Power BI to transform raw data into meaningful insights through data cleaning, modeling, and visualization.

## 🧹 Data Preparation

* Cleaned and transformed data using Power Query
* Removed null values and corrected data types
* Performed data aggregation (Sum of Sales, Profit calculations)

## 📊 Data Modeling & DAX

* Created measures using DAX:

  * Total Sales
  * Total Profit
  * Profit Margin
* Built relationships between tables

## 📈 Visualizations

* Bar Chart – Sales by Region
* Column Chart – Profit by Product
* Pie Chart – Sales Contribution by Region
* Line Chart – Sales Trend over Time
* KPI Cards – Total Sales, Total Profit, Profit Margin

## 🎛️ Features

* Interactive slicers (Region, Product)
* Dynamic filtering for better analysis
* Clean and user-friendly dashboard design

## 🛠️ Tools Used

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
